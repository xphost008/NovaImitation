# PCL II Nova App


下一代 PCL 启动器，基于官方版 PCL2 的完全重构\
采用 Web 技术和内存安全的 Go 全新构建\
这，就是 PCL II: Nova！


第一个可用（功能有限）发行版计划于 2025.7 之前推出\
在此之前你可以自行下载代码编译运行\
请确保安装 Node 21+ 和 GoLang\
你可以参阅[该网址](https://wails.io/docs/gettingstarted/installation)初始化项目，随后构建~
